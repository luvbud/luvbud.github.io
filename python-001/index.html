파이썬 6시간 공부해보자 (1) - 자료형

Posted at 2020-04-10 study python 


섹션 1. 자료형


1. 숫자 자료형

▷ 입력
# 숫자 자료형
print(5)
print(-10)
print(3.14)
print(1000)
print(5+3)
print(2*8)
print(3*(3+1))
출력
5
-10
3.14
1000
8
16
12



2. 문자 자료형

▷ 입력
# 문자 자료형
print('풍선')
print("나비")
print("ㅋㅋㅋㅋ")
print("ㅋ"*4)
출력
풍선
나비
ㅋㅋㅋㅋ
ㅋㅋㅋㅋ



3. boolean 자료형

▷ 입력
# boolean 자료형
print(5>10)
print(5<10)
print(True)
print(False)
print(not True)
print(not False)
print(not(5>10))
출력
False
True
True
False
False
True
True



4. 변수

▷ 입력
# 변수
animal = "강아지"
name = "따용"
age = 7
hobby = "산책"
is_adult = age >= 3

print("우리집 " + animal + " 이름은 " + name)
hobby = "인형물기"
print(name + "이는 " + str(age) + "살이며, " + hobby + "을 좋아함")
print(name, "이는 ", age, "살이며, ", hobby ,"을(를) 좋아함")
print(name + "이는 어른일까? " + str(is_adult))
출력
우리집 강아지 이름은 따용
따용이는 7살이며, 인형물기을 좋아함
따용 이는 7 살이며, 인형물기 을(를) 좋아함
따용이는 어른일까? True



5. Quiz

▷ 입력
# Quiz 1.
station = "사당"
print(station + "행 열차가 들어오고 있습니다.")

station = "신도림"
print(station + "행 열차가 들어오고 있습니다.")

station = "인천공항"
print(station + "행 열차가 들어오고 있습니다.")
출력
사당행 열차가 들어오고 있습니다.
신도림행 열차가 들어오고 있습니다.
인천공항행 열차가 들어오고 있습니다.
