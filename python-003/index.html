2. 슬라이싱

▷ 입력
# 슬라이싱
young = "990105-2311111"
print("성별 : " + young[7])
print("연 : " + young[0:2]) # 0부터 2직전까지 (0, 1)
print("월 : " + young[2:4])
print("일 : " + young[4:6])
print("생년월일 : " + young[:6]) # 처음부터 6직전까지
print("뒷자리 : " + young[7:]) # 7부터 끝까지
print("뒷자리 (뒤부터) : " + young[-7:]) # 맨 뒤에서 7번째부터 끝까지
출력
성별 : 2
연 : 99
월 : 01
일 : 05
생년월일 : 990105
뒷자리 : 2311111
뒷자리 (뒤부터) : 2311111

6. Quiz

▷ 입력
# Quiz 3.
# 사이트별로 비밀번호를 만들어주는 프로그램을 작성하시오
# 예) http://naver.com
# 규칙1: http:// 부분은 제외 => naver.com
# 규칙2: 처음 만나는 점(.) 이후 부분은 제외 => naver
# 규칙3: 남은 글자 중 처음 세자리 + 글자 갯수 + 글자 내 'e' 갯수 + '!' 로 구성
#                         (nav)                       (5)                    (1)                  (!)
# 예) 생성된 비밀번호 : nav51!

# 내가 작성한 코드
site = "http://naver.com"
site = site[7:site.index(".")]
print("생성된 비밀번호 : " + site[:3] + str(len(site)) + str(site.count("e")) + "!")

# 답
url = "http://google.com"
my_str = url.replace("http://", "") # 규칙1
my_str = my_str[:my_str.index(".")] # 규칙2
# my_str[0:5] -> 0 ~ 5 직전까지 (0, 1, 2, 3, 4)
password = my_str[:3] + str(len(my_str)) + str(my_str.count("e")) + "!"
print("생성된 비밀번호 : {0}".format(password))
출력
생성된 비밀번호 : nav51!
생성된 비밀번호 : goo61!
